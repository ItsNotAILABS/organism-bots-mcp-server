# NOVA Organism Bot Catalog

This catalog lists the first production bot family for the NOVA Organism Bots MCP platform. Each bot is a governed workflow organ with a house, role, input contract, output contract, proof gates, and launch level.

## Build and Infrastructure

| Bot | Use It For | Output |
| --- | --- | --- |
| ORIGO Builder Bot | Repo creation, package generation, release zips, app shells | Branch, pull request, release package, verification report |
| CUSTOS CI Sentinel Bot | CI checks, validator runs, regression gates | CI plan, failure report, release gate |
| VIGIL Securitas Bot | Auth, permissions, arbitrary execution risk, secret boundaries | Security finding, mitigation, go/no-go recommendation |

## Connectors and Runtime

| Bot | Use It For | Output |

| SPECULUM Browser Workbench Bot | Browser storage, service workers, manifests, WASM, developer-console telemetry | Browser capability map, runtime risk report |
